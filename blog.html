Question: What is the Document Object Model (DOM)?

Answer: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each object corresponds to a part of the document, such as an element, attribute, or text.

Question: How do you select an element from the DOM?

Answer: There is many ways that i can select an element from the DOM, but the most common one is by using the getElementById(), we can also select a element by using the querySelector() and getElementByClassName()

Question: What is event delegation in the context of the DOM, and why is it useful?

Answer: Event delegation in the context of the DOM refers that how to handling events in top to bottom, manner by assigning a single event listener to a parent element, rather than adding individual event listeners to multiple child elements. This approach takes advantage of event bubbling, where events propagate up from the target element to its ancestors in the DOM tree

Question: How do you manipulate an element's attributes and styles using the DOM?

Answer: To change an element's attributes or styles, we can use JavaScript's Document Object Model (DOM). For attributes, use getAttribute to get their values, setAttribute to set new values, and removeAttribute to remove them. For styles, use style.propertyName.
